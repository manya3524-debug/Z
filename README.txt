Upload these files directly to the root of your GitHub repository:
- index.html
- style.css
- script.js

Then:
Settings > Pages > Deploy from a branch > main > /(root) > Save

Important:
The racing car is an original F1-inspired CSS illustration, not an official Red Bull logo/car.

To add music:
1. Add an MP3 file to the repository, for example music.mp3
2. In index.html find:
   <source src="" type="audio/mpeg">
3. Change it to:
   <source src="music.mp3" type="audio/mpeg">
